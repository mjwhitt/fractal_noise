# perlin_noise

simple ruby script for generating images of 2D perlin noise

**Installation**

1. git clone git@github.com:mjwhitt/perlin_noise.git
2. cd perlin_noise/
3. bundle install --path vendor/
4. ./bin/perlin_noise --help

**Example Output**

./bin/perlin_noise -s 1138 -w 256 -h 256 -n
